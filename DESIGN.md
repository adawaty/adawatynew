# Design Brainstorm (Adawaty)

## Approach A — “Neo-Brutalist Systems Console” (p=0.22)
- **Design movement:** Neo-brutalism + OS console aesthetics
- **Core principles:** hard edges, bold labels, loud hierarchy, utilitarian grids
- **Color philosophy:** stark black/white with one aggressive accent (acid teal)
- **Layout paradigm:** broken grid, chunky blocks, large typographic labels
- **Signature elements:** boxed tags, stamped section headers, “system status” chips
- **Interaction:** clicky buttons, snap transitions, tactile hover offsets
- **Animation:** quick slide snaps, strong easing, section reveals like panels opening
- **Typography:** display: Space Mono / body: IBM Plex Sans

## Approach B — “Dark Swiss Modernism, Warm Accents” (p=0.41)
- **Design movement:** Swiss/International style adapted to dark UI
- **Core principles:** clarity, rhythm, restrained elegance, typographic precision
- **Color philosophy:** deep ink backgrounds for trust; teal for action; amber for warmth
- **Layout paradigm:** disciplined columns + occasional asymmetric hero overlap
- **Signature elements:** ultra-clean cards, thin rules, oversized H1 with tight tracking
- **Interaction:** subtle but confident (underline sweeps, focus halos)
- **Animation:** staggered fade-up on load, minimal parallax in hero image
- **Typography:** display: Sora / body: Source Sans 3

## Approach C — “Cairo Circuit Futurism” (p=0.17)
- **Design movement:** Futurist tech posters + circuit-diagram motifs
- **Core principles:** cinematic contrast, diagonal flow, layered glow, memorable silhouettes
- **Color philosophy:** night-sky navy + luminous teal for ‘data’; amber sparks as ‘human warmth’
- **Layout paradigm:** diagonal section cuts, overlapping panels, ‘data-line’ connectors
- **Signature elements:** circuit-line dividers, diagonal section edges, ‘signal’ badges
- **Interaction:** dramatic hover glows, magnetic CTA, scroll-triggered reveals
- **Animation:** cinematic hero reveal, moving gradient mesh, gentle drifting ‘nodes’
- **Typography:** display: Oxanium / body: Fraunces? (too editorial) → body: Onest (clean)

## Selected direction
**We will commit to Approach C — “Cairo Circuit Futurism”.**

Reasons:
- Differentiates Adawaty from generic consulting templates.
- Matches AI/automation narrative while keeping credibility via disciplined spacing.
- Creates a recognizable visual system we can reuse across pages.
